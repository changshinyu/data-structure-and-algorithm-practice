# Infix / Postfix / Prefix Converter

A **C++/CLI Windows Forms application** that allows users to convert between **infix, postfix, and prefix expressions**.  
It supports basic arithmetic operators (`+`, `-`, `*`, `/`, `^`) and logical operators (`&`, `|`), and provides a GUI to input expressions and display results.

---

## Features

- Convert **Infix → Postfix**
- Convert **Infix → Prefix**
- Convert **Postfix → Infix**
- Convert **Prefix → Infix**
- Convert **Prefix → Postfix**
- Display conversion results in a `RichTextBox` for clarity
- Error handling for invalid expressions
- Step-by-step operand/operator handling using internal stacks
