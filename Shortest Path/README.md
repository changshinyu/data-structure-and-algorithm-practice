# Graph Algorithms GUI (C++/CLI)

This project implements graph algorithms with a Windows Forms GUI in C++/CLI.  
It includes:

- Single Source Shortest Path (Dijkstra)
- All Pairs Shortest Path (Floyd-Warshall)
- Transitive Closure
- Graph generation (random) and file input
- Visualization in DataGridView and TextBox output

## Features

- Supports both directed and undirected graphs
- Adjustable weights and infinite values
- Tab control for different algorithm results
- Optional text output to TextBox for logs

## Usage

1. Open in Visual Studio.
2. Create a Windows Forms project.
3. Replace the Form code with `GraphForm.cpp`.
4. Build and run.
5. Use the buttons to load a file, generate a random graph, or run algorithms.
