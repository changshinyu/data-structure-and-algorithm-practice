# Magic Square Generator (C++/CLI)

This project is a Magic Square generator built with C++/CLI and Windows Forms.  
It can generate magic squares for odd numbers n and for n = 4, and supports different starting positions and directions.

## Project Files

- **MagicSquare.h / MagicSquare.cpp** : Magic square generation algorithms
- **MainForm.h** : GUI form and controls
- **Main.cpp** : Program entry point that launches the GUI

## How to Use

1. Open the project using Visual Studio  
2. Build and run the program  
3. Enter the value of n in the GUI  
4. Select the starting position and direction  
5. Click the generate button  
6. The result will be displayed in the ListBox and DataGridView

## Features

- Supports magic squares for odd n
- Supports magic square for n = 4
- Selectable starting direction  
  (Upper Left / Upper Right / Lower Left / Lower Right)
- Displays the magic square in a DataGridView grid
