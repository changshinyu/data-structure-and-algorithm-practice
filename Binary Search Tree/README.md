# Binary Search Tree GUI (C++/CLI)

This project demonstrates a Binary Search Tree (BST) implemented in C++/CLI with a Windows Forms GUI.  
It supports:

- Insert, Delete, Search
- Recursive & Non-Recursive Traversals (Inorder, Preorder, Postorder, Level-order)
- Visual drawing of the BST
- Clearing the tree

## Features

- **BST Operations**: Recursive and iterative traversals
- **Visualization**: Nodes drawn in a panel, with edges and values
- **Interactive**: Insert values, delete nodes, traverse tree, visualize

## Usage

1. Open in Visual Studio.
2. Create a Windows Forms project and replace `Form.cpp` content with `BSTForm.cpp`.
3. Build and run.
4. Use text boxes and buttons to interact with the BST.
