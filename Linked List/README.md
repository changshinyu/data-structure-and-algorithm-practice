# Linked List (Singly, Doubly, Stack, Queue)

A **C++/CLI Windows Forms application** for practicing and visualizing common **linked structures**:

- **Singly circular linked list**
- **Doubly circular linked list**
- **Stack (linked list implementation)**
- **Queue (linked list implementation)**

It provides **GUI controls** to insert, delete, search, reverse, and display data.

---

## Features

### Singly Circular Linked List
- Insert first / last
- Insert after / before a target node
- Delete after a target node
- Search for nodes
- Print and reverse the list
- Generate random nodes

### Doubly Circular Linked List
- Insert first / last
- Insert after a target node
- Delete after a target node / delete a node
- Search for nodes
- Print forward and backward
- Reverse the list
- Generate random nodes

### Stack
- Push and pop
- Display stack
- Handle empty stack

### Queue
- Enqueue and dequeue
- Display queue
- Check if empty
